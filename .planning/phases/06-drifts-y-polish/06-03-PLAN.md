    **Estrategia recomendada (idempotente y segura):** ejecutar este script Python de un solo uso en host (no requiere DB ni Docker — pure file IO):

    ```bash
    python3 - <<'PY'
    from pathlib import Path

    p = Path("backend/routes/games_routes.py")
    src = p.read_text().splitlines(keepends=True)

    # Colapsar runs de 3+ líneas vacías consecutivas a exactamente 2.
    out = []
    blank_run = 0
    for line in src:
        if line.strip() == "":
            blank_run += 1
            if blank_run <= 2:
                out.append(line)
            # blanks 3 y siguientes se descartan
        else:
            blank_run = 0
            out.append(line)

    # Quitar también la(s) línea(s) en blanco final(es) más allá de UNA al final del archivo
    # (POSIX text-file convention: termina con un único \n).
    while len(out) >= 2 and out[-1].strip() == "" and out[-2].strip() == "":
        out.pop()

    p.write_text("".join(out))
    print("OK: collapsed triple-blank runs to 2; trimmed trailing blank lines.")
    PY
    ```
